Hokuto No Ken
===========

![Screenshot](screenshot.jpg)

Small and old homebrew Lua game written in Lua for PSP 


##What's this ?
Move Ken using directions to punch enemies.<br/>
And enjoy!

##Notes
Compatible with PSP Fat and Slim

##Installation Path
ms0:/PSP/GAME/
